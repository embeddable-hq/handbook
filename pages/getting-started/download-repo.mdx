<Pill>Takes ~3 mins</Pill>

<Steps>

### Download our Boilerplate Repo

Download or clone our boilerplate repo [here](https://github.com/embeddable-hq/remarkable-pro-boilerplate). This includes the **full suite** of **Remarkable Pro components**, our production-ready component library. 

```bash
git clone https://github.com/embeddable-hq/remarkable-pro-boilerplate
```
### Install Dependencies

Navigate into the cloned directory and install the required dependencies:

```bash
npm i
```
### Set Your Region

In your `embeddable.config.ts` file, depending on [your region](/deployment/deployment-regions), uncomment either:
```js
region: 'US',
```
or
```js
region: 'EU',
```

### Build the Code Bundle

Run the following command to build your code bundle:

```bash
npm run embeddable:build
```
