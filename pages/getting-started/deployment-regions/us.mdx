# US Deployment

Once set up on a US workspace, you'll log into Embeddable at [https://app.us.embeddable.com](https://app.us.embeddable.com).

### Update Your Configuration

In your code repo, you'll need to update your `embeddable.config.js` file ([here](https://github.com/embeddable-hq/vanilla-components/blob/main/embeddable.config.js)) to look like this:

```js
import { defineConfig } from '@embeddable.com/sdk-core';
import react from '@embeddable.com/sdk-react';

export default defineConfig({
  plugins: [react],
  previewBaseUrl: 'https://app.us.embeddable.com',
  pushBaseUrl: 'https://api.us.embeddable.com',
  audienceUrl: 'https://auth.us.embeddable.com',
  authDomain: 'auth.us.embeddable.com',
  authClientId: '6OGPwIQsVmtrBKhNrwAaXhz4ePb0kBGV',
});
```

### Web Component Example

The Web Component should look like this:

```js
<em-beddable 
  base-url="https://api.us.embeddable.com/"
  token="YOUR_TOKEN"
/>
```

Learn more [here](/deployment/embedding).

