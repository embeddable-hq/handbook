# Quick-start guide

This step-by-step guide walks you through the **core concepts** of Embeddable in just 20-25 minutes.

<ImageGrid images={["/img/dash3.png"]}/>

## Prerequisites

Before starting on this guide, make sure you have the following software installed:

- A code-editor suitable for JavaScript such as [VsCode](https://code.visualstudio.com/) or [SublimeText](https://www.sublimetext.com/).
- The [NodeJS](https://nodejs.org/en/download) version 20 or newer.
- The version-control system [Git](https://git-scm.com/install/).

This guide also assumes a basic understanding of the programming language JavaScript.
