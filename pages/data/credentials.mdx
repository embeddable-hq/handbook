# Database credentials

Under the hood, Embeddable relies on an open-source library called [Cube](https://cube.dev/docs/product/configuration/data-sources) to connect to your database.

Below are the `type` and `credentials` fields youâ€™ll need when [connecting your database](./connect-your-database).

### Athena
[Cube docs](https://cube.dev/docs/product/configuration/data-sources/aws-athena)
```javascript
type: 'athena',
credentials: {
  accessKeyId: '...',
  secretAccessKey: '...',
  region: '...',
  // S3OutputLocation: '...',
  // workGroup: '...',
  // catalog: '...',
  // schema: '...',
  // concurrency: '...',
}
```

### BigQuery
[Cube docs](https://cube.dev/docs/product/configuration/data-sources/google-bigquery)
```javascript
type: 'bigquery',
credentials: {
  projectId: '...',
  credentials: {
    "type": "service_account",
    "project_id": "...",
    "private_key_id": "...",
    "private_key": "...",
    // ...
  }
}
```

### Clickhouse
[Cube docs](https://cube.dev/docs/config/databases/clickhouse)
```javascript
type: 'clickhouse',
credentials: {
  database: '...',
  host: '...',
  user: '...',
  password: '...',
  port: 8443
}
```

### Databricks
[Cube docs](https://cube.dev/docs/product/configuration/data-sources/databricks-jdbc)
```javascript
type: 'databricks-jdbc',
credentials: {
  url: '...',
  token: '...',
}
```

### MySQL / MariaDB
[Cube docs](https://cube.dev/docs/config/databases/mysql)
```javascript
type: 'mysql',
credentials: {
  database: '...',
  host: '...',
  user: '...',
  password: '...',
  port: 3306,
  ssl: true
}
```

### Postgres
[Cube docs](https://cube.dev/docs/config/databases/postgres)
```javascript
type: 'postgres',
credentials: {
  database: '...',
  host: '...',
  user: '...',
  password: '...',
  port: 5432,
  ssl: true // or { rejectUnauthorized: false }
}
```

### MS SQL / SQL Server
[Cube docs](https://cube.dev/docs/config/databases/mssql)
```javascript
type: 'mssql',
credentials: {
  server: '...',
  port: 1433,
  user: '...',
  password: '...',
  database: '...',
  options: {

  }
}
```
